# デジタル推進委員取組ご協力企業・団体等一覧 (非公式)

デジタル庁が公開している「[デジタル推進委員の取組にご協力いただいている企業・団体等の一覧](https://www.digital.go.jp/policies/digital_promotion_staff/cooperation)」をもとに作成した、検索・ソート機能付きの非公式Webページです。

## 特徴

*   **検索機能**: 企業・団体名やURLでリアルタイムに検索・絞り込みが可能です。
*   **都道府県フィルタ**: 所在都道府県で一覧を絞り込むことができます（ドロップダウンは北から南への地理順）。
*   **ソート**: デフォルトで企業・団体名順にソートされて表示されます。
*   **レスポンシブデザイン**: PC、タブレット、スマートフォンで見やすく表示されます。
*   **ローカル実行**: Webサーバーを必要とせず、ファイルをダウンロードしてそのままブラウザで開くだけで動作します。

## 使い方

1.  このリポジトリをクローンまたはダウンロードします。
2.  `index.html` をお使いのWebブラウザ（Chrome, Edge, Safari, Firefoxなど）で開いてください。

## データについて

`data.js` に含まれるデータは、デジタル庁のWebページから抽出・加工したものです。
利便性向上のため、以下の情報を独自に付与・加工しています：
*   **読み仮名 (kana)**: ソート用
*   **都道府県 (prefecture)**: フィルタリング用（「全国」等の表記を、本社所在地と思われる具体的な都道府県に置き換えている場合があります）
